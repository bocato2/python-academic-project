# Academic Data Analysis with Python

Projeto desenvolvido para prática de manipulação de dados utilizando Python e arquivos CSV.

Todos os dados neste repositório são fictícios e gerados apenas para fins educacionais.
Nomes, e-mails e números de telefone são gerados aleatoriamente.

O sistema simula um ambiente acadêmico contendo dados de:

- alunos
- cursos
- notas
- mensalidades

## Dataset

O dataset contém aproximadamente **1000 alunos** e simula:

- matrícula de alunos
- cursos
- notas em 4 bimestres
- mensalidades de janeiro a dezembro de 2026

## Tecnologias

- Python
- CSV
- DictReader / DictWriter

## Objetivo

Praticar:

- leitura e escrita de CSV
- manipulação de dicionários
- análise de dados
- geração de relatórios

## Possíveis análises

- média de notas por curso
- ranking de alunos
- inadimplência financeira
- estatísticas acadêmicas
